## 📌Project  Overview
This project analyzes **Amazon Bestseller Books data** using **PySpark**, focusing on uncovering patterns in:

- Book ratings  
- Pricing trends  
- Genre performance  
- Author popularity  

Built using **distributed data processing**, the project demonstrates how large datasets can be efficiently analyzed using PySpark.

---

## 🎯Project Objective
- Understand factors influencing bestseller success  
- Analyze relationships between **price, rating, and reviews**  
- Identify top-performing **authors and genres**  
- Apply **PySpark for scalable data analysis**  

---

## ⚙️ Tech Stack
- Python  
- PySpark  
- Pandas (for small transformations)  
- Matplotlib / Seaborn (visualization)  

---

## 📂 Dataset
Amazon Bestsellers dataset containing:

- `Name` – Book title  
- `Author` – Author name  
- `User Rating` – Average rating  
- `Reviews` – Number of reviews  
- `Price` – Book price  
- `Year` – Year of ranking  
- `Genre` – Fiction / Non-Fiction  

---

## 🔄 Workflow

### 1️⃣ Data Loading (PySpark)
- Loaded dataset into Spark DataFrame  
- Handled schema and data types  

### 2️⃣ Data Cleaning
- Checked missing values  
- Standardized columns  
- Converted numeric fields  

### 3️⃣ Exploratory Analysis
- Distribution of ratings and prices  
- Review count trends  
- Genre-wise comparisons  

### 4️⃣ Aggregations (PySpark Strength)
- Top authors by number of bestsellers  
- Average rating by genre  
- Price trends across years  

### 5️⃣ Visualization
- Rating distributions  
- Price vs reviews relationships  
- Genre comparisons  

---

## 📈 Key Insights
✔️ High-rated books are not always the most reviewed  
✔️ Non-fiction dominates in consistent bestseller presence  
✔️ Certain authors repeatedly appear in bestseller lists  
✔️ Price does not strongly correlate with rating  
✔️ Popular books often cluster in moderate price ranges
